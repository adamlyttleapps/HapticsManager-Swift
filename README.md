# HapticsManager-Swift

A simple class for SwiftUI haptics

**Usage:**

```
let hapticsManager = HapticsManager()
hapticsManager.singleTap(.light)
```

**Options:**

```
.soft
.light
.medium
.heavy
.rigid
```
